Module TomlType

module TomlType: sig .. end
Toml types used by parser
type tomlNodeArray =
| NodeEmpty
| NodeBool of bool list
| NodeInt of int list
| NodeFloat of float list
| NodeString of string list
| NodeDate of Unix.tm list
| NodeArray of tomlNodeArray list
Toml arrays are mplemented as lists. Typed, but can contain multiple types if you nest tables
type tomlValue =
| TBool of bool
| TInt of int
| TFloat of float
| TString of string
| TDate of Unix.tm
| TArray of tomlNodeArray
| TTable of tomlTable
Toml primitive values
type tomlTable = (string, tomlValue) Hashtbl.t
Toml table. Implemented as hash table.
